Readme

This repository is my Second Brain. All the issues and improvements to the Second Brain are documented in the notes themselves. This improves portability to other platforms rather than be dependent on GitHub. It also makes the reader aware that a note is possibly not finished. For more guidelines see maintaining.

I use Debian Linux as my operating system. The Linux workflow explains why. Technical note are written for Debian Linux, and do not take Microsoft Windows or macOS into account. They also do not take Arch or Red Hat based distributions into account.
